# 政府推行文物保育的原因：
- 改善國際形象（軟實力 綜合國力）
  - 中國：改革開，GDP—硬實力
- 提升文化質素 文明旅遊
- 拖動經濟發展（旅遊業相關行業）
- 加強凝聚力—提高身分認同
- 回應市民訴求（香港）

# 中國政府推行保育工作的困難
- 保育資金不足
- 配合城市發展需要而清拆，建築難以保留
- 民衆保育意識薄弱令文物受到破壞\
e.g.`長城：XXX到此一游，觸摸`
- 過度參觀，令文物出現超負荷\
e.g.`長假期`
- 地方政府輕視文物保育，損害古跡風貌
- 環境污染令文物受到破壞\
e.g.`盧溝橋-酸雨`
- 保育技術不足
- 走私文物

# 城市發展應該優先於保育工作的原因
- 城市發展能推動經濟增長，維持競爭力
- 城市發展能提升市民的生活素質
- 城市發展可以為保育增加更多資源
# 城市發展不應該
- 古跡文物難以修復，但城市發展可以透過其他方式達到\
e.g.`圍村-丁屋（男丁）-太公分豬肉，新年-族譜`
- 保育工作能提升市民的生活素質、歸屬感\
e.g.`屏山文物徑-食盤菜-祠堂門口-大空地共用`
- 保育工作可以推動經濟發展\
e.g.`日本-祭典`

# 城市發展不應該優先於保育工作的原因
- 古跡文物難以修復，但城市發展可以透過其他方式達到\
e.g. `電子道路收費計劃\赤柱美利樓`

- 保育工作可以推動經濟發展\

# 加强保育的建議(方法)
- 市民關注古跡現狀及表達保育訴求
- 政府增撥保育資源
- 政府完善保育資源
- 政府避免過度開發,符合可持續發展原則
- 保育團體加强宣傳及教育

# 申請世界遺產有利中國文物保育
- 申遺帶來旅遊收益，令地方有更多資源進行保育。\
e.g. `導賞團`
- 申遺令文物變得有名，經過傳媒的大肆報導，讓中國國民加強保育意識。
- 申遺成功後，受聯合國教科文組纖監察，政府為了避免受警告，甚至除名，會加強管理，達致保護文物。
# 申請世界遺產不利中國文物保育
- 申遺成功令該地區吸引大量遊客，中國旅客的不文明行為會加快破壞文物。
- 申遺成功後，旅客增加，加快污染（如交通變得頻繁、大量工程出現），加劇酸問題，破壞文物。
- 申遺成功後商機湧現，使景區出現大量商業活動，變得商業化，原有風貌失真。
